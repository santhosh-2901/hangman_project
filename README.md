# Hangman Game Solver

This project implements a Hangman game solver that combines advanced deep learning models (LSTM and future integration with RNN) and traditional N-gram approaches to optimize letter prediction.

### Prerequisites
- Python 3.x
- TensorFlow
- NumPy
- Requests library
